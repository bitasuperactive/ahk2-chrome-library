◆ __New()

Chrome::__New ( ChromePath : = "",
DebugPort : = 9222,
ProfilePath : = "",
Flags : = "",
URLs : = "" )
inline

Inicia o conecta una instancia de Google Chrome en modo depuración y abre las URL(s) indicadas si no están ya abiertas.

Parámetros
{String}ChromePath (Opcional) Ruta completa al ejecutable de Chrome. Si se deja en blanco, se buscará en los accesos directos, registros del sistema y en la ruta estándar.
{Integer}DebugPort (Opcional) Puerto local para el modo depuración. Por defecto: 9222.
{String}ProfilePath (Opcional) Ruta al perfil de usuario de Chrome. Si no se establece, se utilizará la ruta estándar.
{String}Flags (Opcional) Banderas adicionales para el lanzamiento de Chrome.
{String|Array<String>}URLs (Opcional) URL o colección de URL(s) a abrir al iniciar o conectar Chrome. Si la url exacta ya se encuentra abierta, no se vuelve a abrir.

Escapa las comillas del texto para la línea de comandos.

◆ GetPageBy()

Chrome::GetPageBy ( Key ,
Value ,
MatchMode : = 'exact',
Index : = 1,
fnCallback?  )
inlineprivate

Devuelve una conexión a la interfaz de depuración de la página que coincida con los criterios proporcionados lista para operar.

Cuando varias páginas coinciden con los criterios, aparecen por orden de apertura más reciente.

Parámetros
{String}Key La clave de la lista de páginas a buscar, como "url" o "title".
{String}Value El valor a buscar en la clave proporcionada.
{String}MatchMode (Opcional) Tipo de búsqueda a realizar. Puede ser: exact, contains, startswith o regex. Por defecto: exact.
{Integer}Index (Opcional) Si varias páginas coinciden con los criterios proporcionados, cuál de ellas devolver. Por defecto: 1.
{Func}fnCallback (Opcional) Función a ejecutar cuando se reciba un mensaje de la página: msg => void.
Devuelve
{Chrome.Page|0} Página que coincide con los criterios o 0 si no se encuentra ninguna.
