JSWrapper::__New ( chromeManager ,
url ,
matchMode : = "exact",
events : = 0 )
inline

Crea una nueva instancia de JSWrapper vinculada a una página de Chrome específica, identificada por su URL.

Nota
Controla el cierre manual o inesperado del proceso de Chrome lanzando un error.
Parámetros
{ChromeV2}chromeManager Instancia del administrador de Chrome.
{String}url Enlace de la página a enlazar o abrir.
{String}matchMode (Opcional) Tipo de búsqueda para el enlace. Puede ser: exact, contains, startswith o regex. Por defecto: exact.
{Func}events (Opcional) Función a ejecutar cuando se reciba un mensaje de la página: msg => void.

◆ Login()

JSWrapper::Login ( selectorUserInput ,
selectorPasswordInput ,
selectorLoginButton ,
selectorErrorOutput ,
userName ,
password  )
inline

Realiza el proceso de login en una página web introduciendo el nombre de usuario y contraseña en los campos correspondientes, haciendo click en el botón de login y verificando si se ha producido algún error de autenticación.

Atención
La contraseña se introducirá en texto plano, por lo que se recomienda utilizar esta función solo en entornos controlados o de prueba.
Parámetros
{String}selectorUserInput Selector del campo de entrada del nombre de usuario, por ejemplo: "#username", ".user-field", etc.
{String}selectorPasswordInput Selector del campo de entrada de la contraseña, por ejemplo: "#password", ".pass-field", etc.
{String}selectorLoginButton Selector del botón de login a hacer click, por ejemplo: "#login-button", ".submit-btn", etc.
{String}selectorErrorOutput Selector del elemento que indica un error de autenticación, por ejemplo: "#error-message", ".login-error", etc.
{String}userName Nombre de usuario a introducir en el campo correspondiente.
{String}password Contraseña a introducir en el campo correspondiente.
Devuelve
{Boolean} true si el login se ha realizado correctamente, o false si se ha producido un error de autenticación.

◆ RetrieveTableData()

JSWrapper::RetrieveTableData ( selector )
inline

Extrae los datos de una tabla HTML y los devuelve como un array de objetos ordenados.

Parámetros
{String}selector Selector de la tabla a extraer, por ejemplo: "#data-table", ".info-table", etc.
Devuelve
{Array<OrObject>} Colección de objetos ordenados con los datos de la tabla, donde cada objeto representa una fila y las propiedades corresponden a las cabeceras de la tabla. Si no hay cabeceras o estas contienen otros objetos como enlaces, se les asigna nombres genéricos como "blank_1", "blank_2", etc.
