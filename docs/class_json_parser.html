Descripción detallada

Utilidad para convertir entre cadenas JSON y estructuras de AutoHotkey.

Soporta:

  • Objetos, arrays y valores primitivos.
  • true, false y null preservando tipos.

Implementa parse() para deserializar y stringify() para serializar.

◆ Parse()

JsonParser::Parse ( json ,
keepbooltype : = false,
as_map : = true )
static

Convierte un texto JSON válido en un objeto AutoHotkey.

Parámetros
{String}json Cadena JSON válida.
{Boolean}keepbooltype (Opcional) Si es verdadero, convierte los valores booleanos en sus respectivos equivalentes JSON. En caso contrario, mantiene su valor nativo de AHK. Por defecto: false.
{Boolean}as_map (Opcional) Si es verdadero, los objetos JSON se convierten en Map. En caso contrario, se convierten en Object. Por defecto: true.
Devuelve
{Array|Map|Object} Estructura equivalente al JSON proporcionado.
Excepciones
{Error}Si el texto JSON está mal formado.

static

Convierte un Array/Map/Object plano de AutoHotkey a texto JSON.

Atención
No soporta la serialización de objetos literales anidados.
Nota
Las claves y propiedades añaden un guión bajo (_) al inicio obligatoriamente.
Parámetros
{Any}obj Valor (normalmente un objeto, collección o mapa) a serializar.
{Integer}expandlevel (Opcional) Profundidad máxima que se expandirá. Por defecto: Expande completamente.
{String}space (Opcional) Añade sangrías o espacios al resultado para mejorar su legibilidad. Por defecto: Dos espacios por nivel.
Devuelve
{String} Texto JSON equivalente al objeto.
