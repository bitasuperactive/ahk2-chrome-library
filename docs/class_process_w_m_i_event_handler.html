Descripción detallada

Clase encargada de gestionar los eventos WMI asociados a la creación y eliminación de un proceso específico.

Esta clase actúa como manejador (event sink) para los eventos enviados por WMI mediante el objeto SWbemSink. Se utiliza junto con ProcessWMIWatcher.
